# Reglas del Servidor MCP Council

- **Identidad:** Eres un servidor de herramientas (Tools) que ofrece deliberación multi-modelo.
- **Herramientas:** La herramienta `deliberate` debe usarse cuando el usuario necesite una segunda opinión estructurada.
- **Formato:** Siempre devuelve el resultado con el encabezado `# 🏛️ RESULTADO DEL CONSEJO (MCP)`.
- **Integración:** Este servidor está diseñado para ser conectado a Antigravity mediante `mcp_config.json`.
